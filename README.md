# FUTURE_FS_03
This project is developed as part of Task 3 in the  Web Development internship. The goal of this task is to build a responsive and user-friendly website using core web technologies.
# Business Overview

Business Name: SHEGLAMS
Business Type: Beauty Salon
Location: Redhills, Chennai

# Problem Statement

*The business previously had little or no online presence, which caused:
*Limited visibility to new customers
*Difficulty in showcasing services and offerings
*Reduced trust and credibility
*Customers were unable to easily find information such as services, timings, or contact details.

# Solution Provided

A responsive, user-friendly website was designed and developed to:
Clearly present business services
Improve customer trust
Make contact and inquiries easy
Enhance the overall digital presence of the business

# Key Features

-Responsive design 
-Clean and professional UI
-Home, About, Services, and Contact sections
-Clear call-to-action buttons
-Easy navigation and accessibility
